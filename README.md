# 📊 Automated Sales Analytics Pipeline

## 🚀 Project Overview
This project builds an end-to-end data pipeline that processes sales data, stores it in PostgreSQL, and visualizes insights using Power BI.

---

## 🧱 Tech Stack
- Python (Pandas)
- PostgreSQL
- Power BI
- Task Scheduler

---

## 🔄 Pipeline Flow
CSV → Python → PostgreSQL → Power BI Dashboard

---

## ⚙️ Features
- Data Cleaning & Transformation
- Aggregation to handle duplicates
- UPSERT logic for database updates
- Automated pipeline using Task Scheduler
- Interactive Power BI dashboard

---

## 📊 Key Insights
- Sales trends over time
- Profitability analysis
- Impact of discount on profit
- Regional performance

---

## 🤖 Automation
The pipeline is automated using Windows Task Scheduler to update data periodically.

---

## 🧠 Learning Outcome
- Data Engineering basics
- ETL pipeline creation
- Data modeling
- Dashboard design
